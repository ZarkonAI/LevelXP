# FitnesRPGame
v1 (4–6 дней): дневник тренировок + RPG таблица (уровень/XP/мышцы) + шаблоны. v2: визуал персонажа (WebApp), графики, расширенная аналитика. v3: калории (TDEE, набор/сушка, цели). v4: дневник питания + меню + шаблоны питания. v5: социальные механики (лидерборды/дуэли/кланы), ивенты.


## Безопасность

1) Никогда не коммитьте `.env` и любые ключи.
2) Supabase service role key хранится только в переменных окружения (или `.env` локально).
3) Если ключ случайно утёк:
   - немедленно сгенерируйте новый в Supabase
   - обновите переменные окружения на сервере
   - перезапустите бота
4) В проде используйте:
   - отдельного пользователя/проект Supabase для продакшна
   - ротацию ключей
   - логирование ошибок без утечки данных пользователей
5) Не логируйте:
   - токены
   - ключи Supabase
   - личные данные пользователя (по возможности)
